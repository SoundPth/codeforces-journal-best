# Линейные алгоритмы
## Два указателя
Задача. Найти 2 часила в отсортированном массиве сумма которых равна k.   
Ставим указатель на начало left, а наконец right.   

##  Префиксные суммы
крч так работает
```cpp
int main() {
    int n;
    cin >> n;
    vector <int> a(n), p(n + 1);
    for (int i = 0; i < n; ++i) {
        cin >> a[i];
    }
    p[0] = 0;
    for (int i = 1; i < n + 1; ++i) {
        p[i] = p[i - 1] + a[i - 1];
    }
}
```
